## 🎨 DESIGN ELEMENTS

### Colors
- **Primary**: Purple (#9333EA)
- **Secondary**: Blue (#3B82F6)
- **Success**: Green (#10B981)
- **Danger**: Red (#EF4444)
- **Background**: Dark navy (#0F172A → #1E293B)
- **Text**: White & light gray

### Effects
- **Glassmorphism**: Semi-transparent backgrounds with blur
- **Gradients**: Smooth color transitions
- **Animations**: Pulse, float, hover lifts
- **Shadows**: Soft purple glows on hover

### Typography
- **Hero Title**: 5rem (80px) → responsive down to 2.5rem
- **Section Titles**: 3.5rem (56px) → responsive
- **Body Text**: 1rem (16px) clean and readable

## 📱 RESPONSIVE DESIGN

**Desktop (1200px+)**:
- 2-column AISRI section
- 3-column protocols grid
- Timeline alternates left/right

**Tablet (768-1199px)**:
- 2-column layouts
- Smaller text sizes
- Cards stack better

**Mobile (< 768px)**:
- Single column everything
- Timeline goes vertical
- Full-width buttons
- Larger touch targets
