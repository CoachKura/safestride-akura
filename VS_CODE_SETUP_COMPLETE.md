## 🔧 TROUBLESHOOTING

### **Port 3000 Already in Use**
```
Ctrl+Shift+P → "🧹 Clean Port 3000"
```

### **Changes Not Showing in Browser**
```
Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
```

### **Git Authentication Failed**
```bash
# In terminal (Ctrl+`)
git config --global credential.helper store
git push  # Will prompt for credentials once
```

### **Extensions Not Installing**
```
1. Check internet connection
2. Restart VS Code
3. Install manually: Ctrl+Shift+X → Search → Install
```

## 💡 PRO TIPS

1. **Multi-cursor editing**: `Alt+Click` to add cursors, edit multiple lines at once
2. **Duplicate line**: `Shift+Alt+Down` to duplicate current line
3. **Move line**: `Alt+Up/Down` to move line up/down
4. **Comment line**: `Ctrl+/` to toggle comment
5. **Zen mode**: `Ctrl+K Z` for distraction-free editing
6. **Sidebar toggle**: `Ctrl+B` to show/hide sidebar
7. **File tree focus**: `Ctrl+Shift+E` to focus file explorer
8. **Search in files**: `Ctrl+Shift+F` to search across all files
9. **Replace in files**: `Ctrl+Shift+H` to find and replace
10. **Emmet shortcuts**: Type `div.container>ul>li*3` then Tab
