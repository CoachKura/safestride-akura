Verification Queries

Run these queries in Supabase to verify everything deployed correctly:

1. Check AISRI Scores:

SELECT 
  athlete_name,
  aisri_score,
  risk_category,
  array_to_string(allowed_zones, ', ') as zones,
  current_phase
FROM public.aisri_athlete_dashboard
ORDER BY aisri_score ASC;

2. Check Training Zones:

SELECT zone_code, zone_name, 
       hr_min_percent || '-' || hr_max_percent || '% max HR' as hr_range,
       min_aisri_score
FROM public.training_zones
ORDER BY sort_order;
