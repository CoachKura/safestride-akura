🚀 VS Code Setup Guide

Complete development environment for Akura SafeStride

⚡ Quick Start (5 Minutes)

1

Open Project in VS Code

cd "E:\Akura Safe Stride\safestride"
code .
2

Install Recommended Extensions

Press Ctrl+Shift+P

Type: "Extensions: Show Recommended Extensions"

Click: "Install All"

3

Start Development Server

Press Ctrl+Shift+P

Type: "Tasks: Run Task"

Select: "🚀 Start Development Server"

4

Open Browser

Navigate to: http://localhost:3000

2. Test Locally
