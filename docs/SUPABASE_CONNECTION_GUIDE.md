#  SafeStride - Supabase Backend Connection Guide

**Quick guide to connect your Flutter app to Supabase database**

---

##  WHAT IS SUPABASE?

**Supabase** is an open-source Firebase alternative that provides:
-  **PostgreSQL Database** - Store user data, workouts, assessments
-  **Authentication** - User login/signup with email/password
-  **Real-time Subscriptions** - Live data updates
-  **Row Level Security** - Secure data access policies
-  **Storage** - File uploads (profile pictures, workout videos)
-  **Auto-generated APIs** - No backend coding required

**Why we use Supabase for SafeStride**:
-  Free tier (up to 500MB database + 2GB bandwidth)
-  No credit card required to start
-  Easy Flutter integration with `supabase_flutter` package
-  Built-in user authentication
-  Real-time GPS data sync between coach and athlete
